Install Map Mod

安装地图插件

需要准备一下文件

  • Unreal Engine Mod Loader    本站下载:v2.2    Github:下载
  • MiniMapMod       本站下载:latest

安装Unreal Engine Mod Loader

  • 1.解压UnrealModLoader_Vx.x.x.zip 你会获得以下目录

UnrealModLoader_Vx.x.x
——Profiles
     |—–省略一堆.profile
——Tools
     |—–AutoInjector
          |—–ModLoaderInfo.ini
          |—–readme.txt
          |—–xinput1_3.dll
——ModModLoaderInfo.ini
——UnrealEngineModLauncher.exe
——UnrealEngineModLoader.dll

  • 2.解压后进入Profiles文件夹内,保留一个文件，其余在该文件夹内都删除。

BasicExampleGame.profile

将BasicExampleGame.profile重新命名为 Longvinter-Win64-Shipping.profile 并编辑该文件为以下内容

#Games Basic Information [GameInfo] UsesFNamePool=1 IsUsingFChunkedFixedUObjectArray=1 IsUsingDeferedSpawn=0

  • 3.返回到解压文件的根目录并运行UnrealEngineModLauncher.exe
  • 4.弹出一个指令窗口后，运行Longvinter游戏
  • 5.待游戏完全加载后，退出游戏，进行mod安装。

MiniMap地图Mod安装

需要完成以上所有步骤才能进行这一步。

  • 1.解压MiniMapMod.zip文件
  • 得到这两个文件

MapMod.dll
map_center.png

  • 2.使用Steam打开Longvinter的游戏目录
    会得到一下目录

Engine
     |—–省略
Longvinter
     |—–省略
Longvinter.exe
Manifest_NonUFSFiles_Win64.txt

  • 3.进入Longvinter\Content\CoreMods 目录下
  • 4.将第一步解压的2个文件直接扔到这个目录下,不要有文件夹的形式。
到此你已经完成了所有mod安装，现在只需要每次先打开UnrealEngineModLauncher.exe,再打开Longvinter游戏即可,游戏正常退出的话,注入器也会一同关闭,Alt+F4可能不会,需要手动关闭一次。

使用说明: 开关窗口使用F1键
